用hugo搭建博客

如何用hugo搭建博客 安装hugo 这是hugo的的官网教程：hugo教程 首先我们需要安装hugo，去hugo的releases页面安卓最新版本，解压以后将其所在的文件路径加到PATH里面，打开终端，运行hugo version查看版本 搭建博客 我们点击官网以后，点击里面的Quick Star,按照官网给的目录依次进行，这里我罗列出官网给的步骤： hugo new site quickstart（例如：xxx.github.io-creator） git init(需要cd quickstart) it submodule add https://github.com/budparr/gohugo-theme-ananke.git themes/ananke echo ‘theme = “ananke”’ » config.toml hugo new posts/my-first-post.md hugo server -D 找到config.toml按下图修改 hugo -D 上传到github 做完这些以后就要在github上创建一个名为[你的用户名.github.io]的仓库，然后进入public文件，以此执行以下代码 git init git add . git commit -m hello //关联远程仓库 git remote add origin git@github.com:xxxx/x.github.io.git git push -u origin master 打开x.github.io的Github Pages即可查看博客
